# 노래 영상 다운로드 하기
원래 유튜브 링크에서 앞에 ss를 붙이면 노래를 다운로드 할 수 있지만, 많은 노래를 다운로드 하기에는 너무 많은 시간이 걸려 찾던 도중 ```pytyube``` 패키지를 찾았다.

```python
from pytube import Youtube

# 영상 정보 가져오기
yt = YouTube(url)
# 오디오만 가져오며 파일 형식은 mp4
streams = yt.streams.filter(only_audio=True, file_extension='mp4')
```

하지만 노래 영상은 보통 플레이리스트에 들어있기 때문에 노래는 플레이리스트, 인터뷰는 일반 영상으로 다운로드 하려고 한다.

```python
from pytube import Playlist
 for video in url:
    yt = YouTube(video)
    streams = yt.streams.filter(only_audio=True,file_extension='mp4')

여기서 최고의 음질을 다운로드 하기 위해 streams 중에서 음성 품질이 가장 좋은 것을 뽑았다.

```python
# 최고 품질 뽑기
max_abr = 0
for stream in streams:
    abr = int(re.findall(r'\d+', stream.abr)[0])
    if abr > max_abr:
        max_abr = abr

max_stream.download(path)
```

이렇게 영상을 다운로드하면 영상 제목이 파일 제목이 되는데, 가끔 빠지는 특수 문자들이 몇 개 있다.  
다운로드 받은 파일들의 특수문자들을 처음부터 제거하고, 공백을 없애기로 했다.

```python
files = glob('./music_download/original_files/*.mp4')
for file in files:
    file_name = os.path.basename(file)[:-4]
    file_name = re.sub(r"[^\uAC00-\uD7A30-9a-zA-Z\s]", "", file_name).replace(' ','').replace('musicdownloadoriginalfiles', '')+'.mp3'
    os.rename(file, self.path+file_name)
```
## mp3 -. wav
그리고 wav 파일만 입력할 수 있기 때문에 **ffmpeg**를 이용해서 mp3 파일을 wav 파일로 변환해 주었다.

```python
infile = self.path+file_name
    outfile = f'./music_download/wav_files/{os.path.basename(file_name)[:-4]}.wav'

# 음성파일 설정하기
diff-svc는 wav, 44100Khz, 3~15초, 모노 음성파일만 넣을 수 있기 때문에 그것에 맞게끔 설정을 해 주어야 한다.

먼저 음성파일을 열고 sample_rate와 채널 수를 확인하고 바꿔준다.

```python
with wave.open(path, 'rb') as audio_file:
    # sr(samplingrate 확인)
    sr = audio_file.getframerate()
    # 채널 수 확인(1=모노)
    channels = audio_file.getnchannels()
    # 비트 수 확인(16비트)
    bit_depth = audio_file.getsampwidth() * 8

    if channels == 2:
        sound = AudioSegment.from_wav(file_path)
        sound = sound.set_channels(1)
        sound.export(file_path, format="wav")
        
    if sr != 44100:
        # 음성 파일 불러오기
        y, sr = librosa.load(file_path, sr=None)

        # 리샘플링
        y_resampled = librosa.resample(y, sr, 44100)

        # 결과 저장하기
        librosa.output.write_wav(file_path, y_resampled, sr=44100)
```

그리고 음성 파일을 3~15초에 맞게 잘라 주어야 한다.
