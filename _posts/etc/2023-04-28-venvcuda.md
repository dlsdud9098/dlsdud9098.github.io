CUDA 설치가 일인데, 이거 이용하면 cuda 설치가 쉬울지도?

### 코드

```bash
conda install -y cuda -c nvidia/label/cuda-11.7.0
conda install -y pytorch=2 pytorch-cuda=11.7 -c pytorch
```

만약 wsl를 이용중이라면,

```bash
export LD_LIBRARY_PATH=/usr/lib/wsl/lib
```

이렇게 환경변수도 추가해준다.
