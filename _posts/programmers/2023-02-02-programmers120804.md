## 💡문제 설명
정수 <code>num1</code>, <code>num2</code>가 매개변수 주어집니다. <code>num1</code>과 <code>num2</code>를 곱한 값을 return 하도록 solution 함수를 완성해주세요.

---


## 🚫제한사항

* 0 ≤ <code>num1</code> ≤ 100


* 0 ≤ <code>num2</code> ≤ 100


---


## 🔢입출력 예


<table><thead><tr><th>num1</th><th>num2</th><th>result</th></tr></thead><tbody><tr><td>3</td><td>4</td><td>12</td></tr><tr><td>27</td><td>19</td><td>513</td></tr></tbody>
</table>

---


## 🔍입출력 예 설명
입출력 예 #1

* <code>num1</code>이 3, <code>num2</code>가 4이므로 3 * 4 = 12를 return합니다.

입출력 예 #2

* <code>num1</code>이 27, <code>num2</code>가 19이므로 27 * 19 = 513을 return합니다.

---

## 💻코드

```python
def solution(num1, num2):
    return num1 * num2
```
