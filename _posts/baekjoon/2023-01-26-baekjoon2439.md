## 문제
첫째 줄에는 별 1개, 둘째 줄에는 별 2개, N번째 줄에는 별 N개를 찍는 문제
하지만, 오른쪽을 기준으로 정렬한 별(예제 참고)을 출력하시오.

## 입력
첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.

## 출력
첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.

## 제한

## 예제 입력 1 복사

    5

## 예제 출력 1 복사

        *

       **

      ***

     ****

    *****

## 힌트

```python
a, b = input().split()
c = input().split()
a = int(a)
b = int(b)

d = []
for i in range(a):
    d.append(int(c[i]))

cnt = 0
for i in range(len(c)):
    if b > d[i]:
        print(d[i], end=' ')

```
