## 문제
n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

## 입력
첫째 줄에 n (1 ≤ n ≤ 10,000)이 주어진다.

## 출력
1부터 n까지 합을 출력한다.

## 제한

## 예제 입력 1 복사

    3

## 예제 출력 1 복사

    6

## 힌트


```python
a, b = input().split()
c = input().split()
a = int(a)
b = int(b)

d = []
for i in range(a):
    d.append(int(c[i]))

cnt = 0
for i in range(len(c)):
    if b > d[i]:
        print(d[i], end=' ')

```
