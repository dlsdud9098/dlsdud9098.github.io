conda activate, create를 사용하는데 오류가 났다.

```bash
source /콘다 설치 경로/etc/profile.d/conda.sh
```

ex)

```bash
source /usr/anaconda3/etc/profile.d/conda.sh
```

그리고 다시 하면 정상 작동한다.
