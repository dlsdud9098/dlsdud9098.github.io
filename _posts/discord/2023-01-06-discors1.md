```python
@bot.command()
async def play(ctx):
    channel = ctx.author.voice.channel
    await channel.connect()
```

이런식으로 음성채널로 입장하는 코드를 썼는데 입장이 안된다.

딱히 오류가 뜨지도 않고 다른 명령어는 된다.

그럴때는 ```PyNaCl``` 모듈을 설치하면 된다.
``` pip install PyNaCl```
